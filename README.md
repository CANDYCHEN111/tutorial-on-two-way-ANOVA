# Two-factor-Anova-Spss-BruceR
This is a tutorial on two-way analysis of variance (ANOVA), including tests for simple effects.

This tutorial includes the following experimental designs:

Two-factor completely randomized experimental design

Two-factor completely within-subjects experimental design

Two-factor mixed experimental design

It also includes the raw data in SPSS, corresponding code, and an example of the Two-factor mixed experimental design using R.
